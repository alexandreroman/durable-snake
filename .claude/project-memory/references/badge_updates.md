# Badge firmware updates URL

Official update / info site for the Temporal Replay
Badge: **https://badge.temporal.io**

This URL is referenced in the badge's
`/badgeInfo.json` (note field) as the place to look
for firmware updates and edit instructions for
on-device profile fields.

Use this as the entry point if the user asks about:

- Updating MicroPython firmware on the badge
- Re-flashing or recovering a bricked badge
- Official documentation about the badge hardware
- Cross-checking that an undocumented behavior is
  intentional vs. a firmware bug
